Dossier FilmProject : Projet Spring API REST
Dossier NewGuys : Projet Ionic front

Comment utiliser :
Télécharger wamp (ou autre mais configuration nécessaire pour les accès base de données dans spring)
Importer le script SQL sur la base de données (en localhost)
Build / Run le projet Spring (si il est hebergé ailleurs qu'en localhost, changer l'url des appels AJAX)
Lancer l'application Ionic (ionic serve --lab)
L'utilisateur par défaut est admin / admin, il est possible de créer un compte
